¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño unidimensional de CSS3 que permite distribuir el espacio entre los elementos de una interfaz y mejorar las capacidades de alineación. Con Flexbox, podemos organizar elementos en filas o columnas de manera eficiente.

Para activar Flexbox en un contenedor, usamos la propiedad display: flex. A partir de ahí, el contenedor se convierte en un flex container y sus hijos directos pasan a ser flex items.

Propiedad (Contenedor)Valores principalesDescripción
displayflex / inline-flexActiva el contexto Flexbox
flex-directionrow / column / row-reverseDefine el eje principal
justify-contentflex-start / center / space-between / space-around / flex-endAlineación en el eje principal
align-itemsstretch / center / flex-start / flex-endAlineación en el eje cruzado
flex-wrapnowrap / wrap / wrap-reversePermite que los items pasen a la siguiente línea
gappx / rem / %Espacio entre flex items
Propiedad (Item)Valores principalesDescripción
flex-grownúmero (0, 1, 2…)Cuánto puede crecer el item
flex-shrinknúmero (0, 1…)Cuánto puede encogerse
flex-basisauto / px / %Tamaño base del item
flexshorthand: grow shrink basisLas tres propiedades juntas
align-selfauto / center / flex-endAlineación individual del item
ordernúmero enteroCambia el orden visual del item
01

Cajas en línea — Primeros pasos con Flexbox

Dificultad: Básico
Teoría del Ejercicio

display: flex — Activando el contenedor

Al agregar display: flex a un elemento padre, todos sus hijos directos se convierten en flex items y se organizan en fila por defecto (flex-direction: row).

La propiedad gap establece el espacio entre cada item sin necesidad de usar márgenes. Es la forma moderna y limpia de separar elementos.

Con justify-content controlamos cómo se distribuyen los items a lo largo del eje principal (horizontal en row). Con align-items controlamos el eje cruzado (vertical).

📋 Enunciado

Creá una página con un contenedor que tenga 3 cajas de colores alineadas horizontalmente. Cada caja debe tener un tamaño fijo y mostrar su número en el centro.

  • Paso 1: Crear el HTML con un div contenedor y 3 divs hijos (.caja)
  • Paso 2: Aplicar display: flex al contenedor
  • Paso 3: Agregar gap: 20px para separar las cajas
  • Paso 4: Centrar las cajas en la pantalla con justify-content: center
  • Paso 5: Centrar el texto dentro de cada caja con Flexbox también
index.html 
<!-- Estructura HTML -->
<body>
  <div class="contenedor">
    <div class="caja caja-1">1</div>
    <div class="caja caja-2">2</div>
    <div class="caja caja-3">3</div>
  </div>
</body>
styles.css 
/* === CONTENEDOR FLEX === */
.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 20px;
  min-height: 100vh;
  background-color: #f5f5f5;
}

/* === CADA CAJA === */
.caja {
  width: 100px;
  height: 100px;
  border-radius: 8px;
  display: flex;          /* Flex también en el item */
  justify-content: center;
  align-items: center;
  color: #fff;
  font-size: 1.5rem;
  font-weight: bold;
}

.caja-1 { background-color: #e63946; }
.caja-2 { background-color: #457b9d; }
.caja-3 { background-color: #2a9d8f; }

02

Barra de Navegación — Distribución espacial

Dificultad: Básico-Intermedio
Teoría del Ejercicio

space-between y alineación compuesta

justify-content: space-between es uno de los valores más usados en Flexbox. Coloca el primer item al inicio y el último al final, distribuyendo el espacio sobrante entre los elementos del medio.

En una navbar típica queremos el logo a la izquierda, los links en el centro y un botón a la derecha. Con Flexbox y space-between esto se logra sin trucos.

Recordá que podés tener Flexbox dentro de Flexbox (anidado). Los links de navegación también forman su propio contenedor flex para separarse con gap.

📋 Enunciado

Construí una barra de navegación horizontal con: logo a la izquierda, 3 links en el centro y un botón "Contacto" a la derecha.

  • Paso 1: Estructura HTML con <nav> conteniendo logo, lista de links y botón
  • Paso 2: Aplicar display: flex y justify-content: space-between al nav
  • Paso 3: Centrar verticalmente con align-items: center
  • Paso 4: Convertir la lista de links también en flex con gap
  • Paso 5: Estilizar logo, links y botón con colores y tipografía
index.html 
<nav class="navbar">
  <div class="logo">MiSitio</div>

  <ul class="nav-links">
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Nosotros</a></li>
    <li><a href="#">Servicios</a></li>
  </ul>

  <button class="btn-contacto">Contacto</button>
</nav>
styles.css 
.navbar {
  display: flex;
  justify-content: space-between; /* ← Clave */
  align-items: center;
  padding: 16px 32px;
  background-color: #16213e;
}

.logo {
  font-size: 1.4rem;
  font-weight: bold;
  color: #f0e040;
}

.nav-links {
  display: flex;   /* Flex anidado */
  list-style: none;
  gap: 24px;
}

.nav-links a {
  text-decoration: none;
  color: #aaa;
  font-size: 0.9rem;
  transition: color 0.2s;
}
.nav-links a:hover { color: #fff; }

.btn-contacto {
  background: #f0e040;
  color: #000;
  border: none;
  padding: 8px 18px;
  border-radius: 4px;
  font-weight: bold;
  cursor: pointer;
}

03

Grilla de Tarjetas — flex-grow y columnas

Dificultad: Intermedio
Teoría del Ejercicio

flex-direction: column y flex-grow

flex-direction: column cambia el eje principal a vertical. Esto es fundamental para armar el interior de una tarjeta donde los elementos se apilan uno sobre otro.

flex-grow: 1 en un item le dice a Flexbox "ocupá todo el espacio disponible que sobra". Si dos items tienen flex-grow: 1, el espacio se divide a partes iguales.

La propiedad shorthand flex: 1 equivale a flex-grow: 1; flex-shrink: 1; flex-basis: 0%. Cuando la usamos en los items de una fila, hace que todos ocupen el mismo ancho disponible.

📋 Enunciado

Creá una sección de 3 tarjetas de productos que se distribuyeran en iguales proporciones. Cada tarjeta tiene: imagen (div de color), título, texto y etiqueta.

  • Paso 1: Contenedor flex con 3 tarjetas hijas
  • Paso 2: Usar flex: 1 en cada tarjeta para repartir el ancho en partes iguales
  • Paso 3: Cada tarjeta es también flex con flex-direction: column
  • Paso 4: El texto descriptivo debe empujar la etiqueta hacia abajo usando flex-grow: 1 en el párrafo
  • Paso 5: Agregar gap, bordes redondeados y sombra a las tarjetas
styles.css 
.grilla {
  display: flex;
  gap: 20px;
  padding: 24px;
  background: #fafafa;
}

.tarjeta {
  flex: 1;           /* igual ancho para todas */
  display: flex;
  flex-direction: column; /* apila los hijos */
  gap: 12px;
  background: #fff;
  border-radius: 12px;
  padding: 16px;
  box-shadow: 0 2px 12px rgba(0,0,0,0.08);
}

.tarjeta-img {
  height: 120px;
  border-radius: 8px;
  background: linear-gradient(135deg, #667eea, #764ba2);
}

.tarjeta-titulo {
  font-weight: bold;
  font-size: 1rem;
}

.tarjeta-texto {
  flex-grow: 1; /* empuja la etiqueta abajo */
  color: #666;
  font-size: 0.85rem;
}

.tarjeta-etiqueta {
  align-self: flex-start;
  background: #f0e040;
  color: #000;
  padding: 4px 12px;
  border-radius: 99px;
  font-size: 0.75rem;
  font-weight: bold;
}

04

Landing Page Básica — Layout completo con Flexbox

Dificultad: Intermedio-Avanzado
Teoría del Ejercicio

flex-wrap y layout de página completa

flex-wrap: wrap permite que los flex items pasen a una segunda fila cuando no caben en la primera. Esto es esencial para diseños que deben adaptarse a distintos anchos.

Para construir una página completa con Flexbox se usa la técnica de columna principal: el <body> o wrapper principal se hace flex-direction: column con min-height: 100vh, y el footer se empuja abajo dando al main flex-grow: 1.

La propiedad flex-basis establece el tamaño base antes de que se apliquen flex-grow y flex-shrink. Combinada con flex-wrap, nos permite definir cuántas columnas queremos: flex-basis: calc(33% - 20px) produce 3 columnas.

📋 Enunciado

Armar una landing page con estructura completa: hero, sección de features (3 ítems en fila) y footer. Todo con Flexbox.

  • Paso 1: Wrapper como flex column para que el footer quede siempre abajo
  • Paso 2: Sección hero con texto centrado (flex + center en ambos ejes)
  • Paso 3: Sección features con flex-wrap: wrap y 3 items usando flex-basis
  • Paso 4: Footer con Flexbox y space-between
  • Paso 5: Agregar order para reordenar visualmente uno de los features
styles.css 
.wrapper {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.hero {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: column;
  min-height: 200px;
  gap: 16px;
  background: #0f172a;
}

.features {
  display: flex;
  flex-wrap: wrap;   /* los items saltan de fila */
  gap: 16px;
  padding: 24px;
  flex-grow: 1;      /* ocupa el espacio entre hero y footer */
}

.feature-item {
  flex-basis: calc(33.33% - 11px); /* 3 cols */
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 8px;
  padding: 20px;
  background: #1e293b;
  border-radius: 8px;
}

/* Reordenar visualmente */
.feature-item:first-child {
  order: 2;
}

.site-footer {
  display: flex;
  justify-content: space-between;
  padding: 16px 24px;
  background: #0a0f1e;
}

05

Dashboard con Sidebar — Maestría en Flexbox

Dificultad: Avanzado
Teoría del Ejercicio

Layouts complejos: sidebar fijo + contenido flexible

El patrón sidebar + contenido es uno de los más usados en aplicaciones web. Se logra con un contenedor flex en fila donde el sidebar tiene un ancho fijo (flex-shrink: 0) y el contenido crece (flex: 1).

flex-shrink: 0 impide que un item se encoja por más que el espacio sea limitado. Esto asegura que el sidebar siempre mantenga su ancho definido, sin importar cuánto contenido haya al lado.

Dentro del área de contenido podemos usar nuevamente flex-wrap: wrap para crear una grilla de tarjetas responsiva. Combinando estos patrones podemos construir interfaces complejas con muy poco código.

TécnicaCSS usadoResultado
Sidebar fijowidth: 240px; flex-shrink: 0No se encoge
Contenido adaptableflex: 1Ocupa el resto
Grilla internaflex-wrap: wrap + flex-basisCols automáticas
Sticky footerflex col + flex-grow en mainFooter abajo siempre

📋 Enunciado

Construí un panel de administración con barra lateral fija de navegación y área de contenido principal que muestra una grilla de estadísticas.

  • Paso 1: Layout raíz flex en fila: sidebar + área principal
  • Paso 2: Sidebar: width: 240px, flex-shrink: 0, flex-direction: column
  • Paso 3: Área principal: flex: 1, flex-direction: column (header + grilla)
  • Paso 4: Header del contenido con justify-content: space-between
  • Paso 5: Grilla de tarjetas de estadísticas con flex-wrap: wrap y flex-basis: calc(50% - 8px)
  • Desafío extra: Agregar un ítem de la grilla con flex-basis: 100% para que ocupe todo el ancho
index.html 
<div class="dashboard">

  <aside class="sidebar">
    <div class="sidebar-logo">Admin</div>
    <nav class="sidebar-nav">
      <a class="nav-item active">Dashboard</a>
      <a class="nav-item">Usuarios</a>
      <a class="nav-item">Reportes</a>
    </nav>
  </aside>

  <main class="contenido">
    <header class="contenido-header">
      <h1>Dashboard</h1>
      <span>Hoy</span>
    </header>
    <div class="stats-grid">
      <div class="stat-card">...</div>
      <div class="stat-card">...</div>
      <div class="stat-card full">...</div>
    </div>
  </main>

</div>

.dashboard {
  display: flex;
  min-height: 100vh;
}

.sidebar {
  width: 240px;
  flex-shrink: 0;     /* no se encoge nunca */
  display: flex;
  flex-direction: column;
  gap: 8px;
  padding: 20px 16px;
  background: #1a1a2e;
}

.nav-item {
  padding: 10px 14px;
  border-radius: 6px;
  color: #888;
  text-decoration: none;
  font-size: 0.9rem;
}
.nav-item.active {
  background: #f0e040;
  color: #000;
  font-weight: bold;
}

.contenido {
  flex: 1;              /* ocupa todo el resto */
  display: flex;
  flex-direction: column;
  gap: 20px;
  padding: 24px;
}

.contenido-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.stats-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}

.stat-card {
  flex-basis: calc(50% - 8px); /* 2 columnas */
  padding: 24px;
  background: #fff;
  border-radius: 10px;
  border: 1px solid #eee;
}

/* Desafío extra: item a ancho completo */
.stat-card.full {
  flex-basis: 100%;
}
